Network Creation and Analysis


Choice of the five Wikipedia pages:

- Internet Addiction Disorder
- DSM-V
- Digital Addict
- Addictive Behavior
- Online Identity

1. See also extraction, starting from the html page
(inspect element) using an url extractor

2. Analysis of Wikipedia pages found from the see also
and deletion of the useless elements

3. See also levels from the five Wikipedia pages and first see also level
extraction using Seealsology

4. .csv opened in Gephi and deletion of the wrong voices
(numbers, id containing “:”)

5. .csv cleaning, only the “in-degree” data
greather than 1 kept (Degree range: 2-17)

6. Algorithm used: ForceAtlas 2, parameters LinLog mode,
scaling 0.35, gravity 0.2, prevent overlap and modularity

7. Map exportation and definition
