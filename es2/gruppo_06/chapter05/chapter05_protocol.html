Introduction

The protocol described allows to extract the first 120 cases of case law,
using the query “File Sharing” –“child pornography”.
The reason is that more than half of the cases are related to the acquisition of child pornography via file sharing, and it is a special case that does not affect the dispute, but concerns another type of virtual crime.

Procotol

Get in incognito mode on scholar.google.com and change this settings:

  • Since 2010
  • Sort of relevance
  • Excluding citation

Research the query “File Sharing” –“child pornography” and scrawl 15 pages (160 results).

Delete non-releated cases (for example: A girl denounces Kodak for using her photo without
her permission).

Fill an excel file with the following entries:

Metadata

Timestamp:
04/12/14 - 05/12/14

Data source:
Google Case law

Tools:
Microsoft Excel
